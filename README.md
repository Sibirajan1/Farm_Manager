# Farm_Manager
  This Software Requirements Specification (SRS) describes the requirements for a web-based Farming Expense Notes Management System, Version 1.0. This application is designed to help farmers, especially those with limited technical or financial calculation skills, systematically record, track, and manage their daily farming expenses such as labour wages, fertilizer costs, and other spending. The system provides simple note-taking with date, topic, description, and price details, auto-calculates totals for each note, and includes alerts and reminders through future markings and notifications. Basic user authentication using a simple code is provided to secure access. The application is designed to be simple, clear, and user-friendly to ensure that farmers can easily manage their expense records and receive timely reminders, supporting better farm management and financial planning.
